# 环境安装 #
- [.NET Core SDK 2.1](https://dotnet.microsoft.com/download)

# 快速开始 #
#### 方式一
- 使用Visual Studio 打开/Trains/Trains.sln，将Trains设置为启动项目。

#### 方式二
- 工程在目录 /Trains/Trains 
- graph.txt为默认输入文件，如要修改默认测试值，请修改文件内容
- 在目录 /Trains/Trains 下打开命令窗口 执行dotnet run
